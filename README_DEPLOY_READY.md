# Deploy-ready holat

Bu papka Railway + Netlify uchun tozalangan:

- `backend/.env` olib tashlangan.
- `frontend/node_modules` olib tashlangan.
- `__pycache__` va local fayllar olib tashlangan.
- Railway backend uchun `backend/railway.json`, `backend/Procfile`, `backend/.env.railway.example` tayyor.
- Netlify frontend uchun root `netlify.toml` va `frontend/.env.netlify.example` tayyor.

Deploy tartibini `DEPLOY_RAILWAY_NETLIFY_STEP_BY_STEP.md` faylidan ko‘ring.
